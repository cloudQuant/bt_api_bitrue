# bt_api_bitrue

Bitrue exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_bitrue
```

## Usage

```python
from bt_api_bitrue import BitrueExchangeDataSpot, BitrueRequestDataSpot
```

## Features

- REST API support
- HMAC-SHA256 authentication (Binance-style)
- Symbol format: `BTCUSDT` (concatenated uppercase)
